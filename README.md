# Financial Modelling using Excel (FME)

This repository provides a comprehensive, topic-wise collection of Financial Modelling using Excel (FME) resources designed for MBA-level finance education.

## Repository Structure

Each topic folder contains the following components:

- **Pre-Learning Material**  
  Conceptual explanations, background theory, and context required before model building.

- **Problem Statements**  
  Structured problem descriptions aligned with classroom and lab sessions.

- **Excel Models**  
  Fully built and functional Excel-based financial models.

- **Model Manuals**  
  Step-by-step instructions enabling students to independently build each model from scratch.

## Intended Audience

MBA students specializing in Finance, instructors teaching financial modelling, and learners seeking structured Excel-based finance applications.

## Usage

This repository is intended for academic learning and instructional purposes. Each topic is designed to be self-contained and modular.
